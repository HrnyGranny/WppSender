How to Use
  1. Configure your message settings above
  2. Click "Generate Script" to create your custom script
  3. Copy the generated code
  4. Open WhatsApp Web in your browser
  5. Open a chat where you want to send messages
  6. Press F12 to open browser Developer Tools
  7. Navigate to the "Console" tab
  8. Paste the code and press Enter
  9. To stop the script, refresh the page
⚠️ Use responsibly. Excessive message sending may result in temporary blocks.
